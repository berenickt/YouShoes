# YouShoes
예담직업전문학교 프로젝트 수제화쇼핑몰


### 팀원
1. 유승우 - 프로필, 결제, 통계, 즐겨찾기, 공지사항 
2. 구교동 - 주문, 후기, 결제내역, 가게정보
3. 권우성 - 예약, 배송 
4. 배광준 - 로그인, 판매회원 관리, 구매자 메인 홈


### 개발 환경
1. HTML
2. CSS
3. JavaScript
4. BootStrap
5. jQuery
6. Ajax
7. Oracle -> MySQL
8. Tomcat 8.5


### 사용 API
1. 달력 API - FullCalendar
2. 통계 API - Chart.js
3. 지도 API - 카카오 지도 API
4. 결제 API - I'mport; (아임포트)
