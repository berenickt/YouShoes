# YouShoes
예담직업전문학교 프로젝트 수제화쇼핑몰

# 팀원 : 구교동, 유승우, 권우성, 배광준

유승우 - 프로필, 결제, 통계, 즐겨찾기, 공지사항 

구교동 - 주문, 후기, 결제내역, 가게정보

권우성 - 예약, 배송 

배광준 - 로그인, 판매회원 관리, 구매자 메인 홈


# 개발 언어
HTML, CSS, JavaScript, BootStrap, jQuery, Ajax, Oracle -> MySQL , Tomcat 8.5

# 사용 API
달력 API - FullCalendar
통계 API - Chart.js
지도 API - 카카오 지도 API
결제 API - I'mport; (아임포트)
